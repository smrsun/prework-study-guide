HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The tag refers only to what is inside the angle brackets

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A link element creates the connection to the external file.
  • A rel attribute specifies the relationship between the current document and linked document resource.
  • A href attribute specifies the location (URL) of the external resource.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git status gives you all neccesary information about current branch
  • git add is to add new files to the staging area
  • git merge is to integrate or merge branches together
  • git diff is used when you need to compare two files or branches before you commit or push
